# MVP Scope

## MVP Goal

Build the smallest useful version of the product that proves whether recruiters will use candidate evidence reports.

## MVP Pages

### Public Pages

- Landing page
- Pricing page
- Request demo page

### Recruiter Pages

- Login
- Company dashboard
- Create job page
- Job criteria review page
- Candidate list page
- Candidate evidence report page
- Human decision page

### Candidate Pages

- Application landing page
- Resume upload page
- Profile links page
- Questionnaire page
- Consent page
- Submit confirmation page

## MVP Features

### Must Have

- Company account
- Recruiter login
- Create job
- Paste job description
- AI job criteria extraction
- Editable job criteria
- Candidate application link
- Resume upload
- Candidate consent
- Job-specific questionnaire
- Resume parsing
- Evidence matching
- Candidate evidence report
- Recruiter notes
- Human decision
- Audit log
- PDF export

### Should Have

- Basic fairness check
- Missing evidence section
- Suggested interview questions
- Candidate report status badges
- Search and filter candidates

### Not In MVP

- Mobile app
- Full ATS
- Job board
- Payment system
- Video interviews
- Background checks
- LinkedIn scraping
- Automatic rejection
- Candidate marketplace
- Recruiter marketplace
- Complex analytics
- Enterprise SSO
