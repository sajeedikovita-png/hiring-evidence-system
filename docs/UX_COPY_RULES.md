# UX Copy Rules

## Core Message

Hire with evidence, not guesswork.

## Product Voice

The voice should be:

- Clear
- Honest
- Professional
- Calm
- Human-centered
- Responsible

## Mandatory Language Rule

Never imply that AI makes final hiring decisions.

The system should always show that the recruiter or hiring manager makes the final decision.

## Words To Use

Use:

- Evidence match
- Job-related evidence
- Needs verification
- Missing evidence
- Suggested interview questions
- Fairness check
- Human review required
- Recruiter notes
- Decision reason
- AI-assisted summary
- Review before deciding

## Words To Avoid

Avoid:

- AI selected this candidate
- AI rejected this candidate
- Best candidate
- Perfect match
- Bias-free
- Guaranteed fit
- Culture fit score
- Personality score
- Auto reject
- AI decision
- Winner
- Bad candidate

## Safer Alternatives

Instead of "candidate score", use "evidence match".

Instead of "weak candidate", use "missing evidence".

Instead of "AI recommends rejection", use "insufficient job-related evidence found".

Instead of "best candidate", use "strongest evidence match".

Instead of "culture fit", use "role-related collaboration evidence".

## Required Disclaimer Text

Use this in report pages:

"AI-assisted analysis. Human review is required before making any hiring decision."

Use this in candidate pages:

"Your information will be used only to assess your application for this role unless you give separate consent."

Use this in recruiter decision pages:

"Final decision must be based on job-related criteria and reviewed by a human."
