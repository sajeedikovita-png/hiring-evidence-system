## Safety/Product Rules

Do not use language like:

- AI selected
- AI rejected
- Best candidate
- Perfect match
- Bias-free
- Auto reject

Use language like:

- Evidence report ready
- Human review required
- Good evidence, verification needed
- Missing evidence
- Needs verification
- Job-related evidence
- Decision reason required
